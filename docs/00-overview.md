# Sistema Cartão Mais Saúde

## Objetivo
Sistema SaaS para gestão de clientes de cartão de benefícios em saúde.

## Funcionalidades principais
- Cadastro de clientes
- Planos com especialidades e carência
- Controle financeiro (mensalidades)
- Validação de elegibilidade
- Carteira virtual com QR Code
- Portal do parceiro

## Público
- Operadora do cartão
- Parceiros (clínicas)
- Clientes

## Diferencial
- Validação em tempo real
- Multi-tenant
- Arquitetura SaaS escalável
